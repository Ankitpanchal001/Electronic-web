# Electronic Web

A responsive and modern static website for showcasing electronic products or services.

🌐 **Live Demo**: [Visit Site](https://ankitpanchal001.github.io/Electronic-web/)

---

## 📁 Project Structure

Electronic-web/
│
├── css/ # Stylesheets
├── js/ # JavaScript files
├── images/ # Image assets
├── icomoon/ # Icon font assets
├── index.html # Main HTML file
├── style.css # Additional styling
└── readme.txt # Original readme or legacy notes
---

## 🚀 Features

- Mobile responsive layout
- Attractive banner sections and product cards
- Icon fonts and image sliders
- Lightweight and fast-loading
- Fully static (HTML, CSS, JS) – No backend required

---

## 📱 Responsive Design

This site is mobile-friendly and works well on all modern browsers and screen sizes.  
Make sure `<meta name="viewport">` is present in `index.html` for optimal scaling.
